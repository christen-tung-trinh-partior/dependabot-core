## `dependabot-nuget`

NuGet support for [`dependabot-core`][core-repo].

### Running locally

1. Start a development shell

  ```
  $ bin/docker-dev-shell nuget
  ```

2. Run tests
   ```
   [dependabot-core-dev] ~/dependabot-core $ cd nuget && rspec
   ```

[core-repo]: https://github.com/dependabot/dependabot-core
